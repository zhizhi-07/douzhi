# 📋 线下模式预设使用教程

## 🚀 快速开始

### 方法一：桌面导入器（推荐）

1. **打开导入器**
   - 双击 `桌面预设导入器.html` 
   - 会在浏览器中打开

2. **导入预设**
   - 将桌面的 `【MoM】果实4.75丨喵喵电波丨0819.json` 拖放到上传区域
   - 或点击上传区域选择文件
   - 支持批量导入多个预设

3. **查看已导入**
   - 导入成功后会显示在列表中
   - 预设已自动保存到浏览器

4. **使用预设**
   - 打开应用，进入任意角色的线下模式
   - 点击右上角📄图标打开预设管理
   - 在列表中选择【MoM】果实4.75丨喵喵电波丨0819
   - 开始对话，AI会使用该预设的叙事风格

---

### 方法二：在线上传

1. 进入线下模式
2. 点击右上角📄图标
3. 在弹出的预设管理窗口中：
   - 点击"上传新预设"按钮
   - 选择预设JSON文件
4. 在列表中点击预设名称切换

## 🎨 自定义预设

### 预设文件格式

```json
{
  "name": "你的预设名称",
  "system_prompt": "你的提示词内容...",
  "description": "预设描述（可选）",
  "temperature": 0.85,
  "top_p": 0.92,
  "max_tokens": 800
}
```

### 关键字段说明

| 字段 | 必需 | 说明 |
|------|------|------|
| name | ✅ | 预设名称，会显示在列表中 |
| system_prompt | ✅ | 系统提示词，定义AI的叙事风格 |
| description | ❌ | 预设描述 |
| temperature | ❌ | 温度参数，控制创造性（注：目前由API设置控制） |
| top_p | ❌ | 采样参数 |
| max_tokens | ❌ | 最大token数 |

### SillyTavern变量支持

预设中可以使用以下变量，系统会自动替换：

- `{{char}}` - 角色名
- `{{user}}` - 用户名
- `{{time}}` - 当前时间
- `{{date}}` - 当前日期

## 💡 预设编写技巧

### 1. 明确叙事视角
```
你是第三人称全知视角的叙事者，以"他/她"称呼角色...
```

### 2. 定义输出格式
```
### 输出格式
- 环境描写：先描绘场景氛围
- 动作描写：具体的肢体语言
- 对话：符合人物性格
- 心理活动：用【】标记
```

### 3. 设置约束条件
```
## 约束
1. 禁止替用户做决定
2. 保持角色人设一致
3. 避免过度文艺腔
4. 合理控制篇幅（150-300字）
```

### 4. 提供示例
```
### 示例输出：
下午三点，卧室。
阳光透过窗帘洒进来...
```

## 🔧 常见问题

### Q: 为什么预设没有生效？
**A:** 
1. 确认已在预设管理中选择（有✓标记）
2. 确认预设文件格式正确
3. 刷新页面重新加载

### Q: 可以有多个预设吗？
**A:** 可以！支持批量导入，随时切换

### Q: 预设会影响线上聊天吗？
**A:** 不会。预设只在线下模式生效

### Q: 删除预设后能恢复吗？
**A:** 不能。删除前请确认

### Q: 预设保存在哪里？
**A:** 保存在浏览器LocalStorage中，清除浏览器数据会丢失
