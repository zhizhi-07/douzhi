## 🎯 设计理念

- **小巧精致**: 应用图标56x56，圆角16px
- **液态玻璃**: 半透明背景 + 高斯模糊 + 边框高光
- **流畅动画**: cubic-bezier缓动，0.3s过渡
- **触摸反馈**: 按下缩放92%，涟漪效果

## 🎨 颜色系统

- 玻璃背景：`rgba(255, 255, 255, 0.15)`
- 模糊强度：`blur(20px)`
- 边框高光：`rgba(255, 255, 255, 0.2)`
- 投影：`0 8px 32px rgba(0, 0, 0, 0.1)`
